ubuntu软件安装说明

发表于 2017-04-04 |

ubuntu 16.04 x86_64 kernel 4.4.0-47-generic

必备软件

chrome 浏览器

adobe flash播放

FireFox ok
Chrome 报错：Adobe Flash Player已过期
解决方法：
step 1，设置全局翻墙代理
step 2，在Chrome的地址栏里面输入，chrome://components/ 然后手动更新adobe flash player插件

视频播放vlc

安装virtualbox 5.1

QQ

优麒麟

开发基础

基础编译工具

apt-get install build-essential

git

sudo apt-get install git

openjdk 8

sudo apt-get install openjdk-8-jdk

sublime text3

中文输入有问题，需要每个人自己解决，参考这里
https://github.com/lyfeyaj/sublime-text-imfix

各种插件需要安装，安装package control需要翻墙，请翻墙后再安装

nodejs 安装

nodejs在线升级脚本：
curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -
等待完成后安装nodejs
sudo apt-get install -y nodejs
安装完成后即可
node -v 查看 node安装版本
npm -v 查看 npm安装版本

npm管理工具更换国内淘宝源以及npm更新

vi ~/.npmrc 编辑.npmrc文件
插入registry = https://registry.npm.taobao.org,输入:wq 退出并保存
cat ~/.npmrc 执行
sudo npm install -g npm 更新npm

数据库安装

sudo apt-get install mysql-server
https://help.ubuntu.com/12.04/serverguide/mysql.html#mysql-installation

teamviewer

https://www.linuxbabe.com/desktop-linux/install-teamviewer-ubuntu-16-04-xenial-xerus

http://ubuntuhandbook.org/index.php/2016/01/install-variety-wallpaper-changer-in-ubuntu-16-04/

XX-Net

使用 xx-NET 配置为 Acquire::https::Proxy “http://127.0.0.1:8087“;

导入xx-net证书
sudo cp ~/tools/XX-Net-3.2.7/data/gae_proxy/CA.crt /usr/local/share/ca-certificates/

sudo update-ca-certificates

sudo apt-get update
